# simple-rest-yii2
Yii2 app with REST API based on parsing special XML.
Sample XML document located at `@webroot/trips.xml`



##Return data

*'error' => string,*
*'roundTrip' => boolean,*
*'residenceSeconds' => $array,*
*'destinationCity' => string,*
*'breakpoint' => string,*
*'boardCities' => array,* 
*'offCities' => array*
